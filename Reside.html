About

Real estate transaction is a complex and time-consuming process in the United States. Reside is an online platform supporting real estate agents by streamlining the transaction process from clients info management, transaction documents generation/signing/management, listing services, marketing services, property analytics report and so on. By standardizing and automating the transaction process to a great extent, it helps agents to close transactions much more efficiently.

As a product designer at Reside, my job included use research, product design, and user testing. I shipped several core features of the platform on both desktop and mobile. Also, I led part of the platform’s design system creation.

Feature - Transaction List View and Filter

Transaction list is the page providing entrance for transactions of the user. It also provides an overview of all transactions including their status and basic information. The transaction filter in this page helps to organize transactions by different attributes including status, client and agent.

Feature - Document Tagging

Document tagging allows users to tag a pdf document with signing fields for different recipients. Then, the document could be sent out to recipients for signature request. Also, preset fields change based on the selection of the recipient.

Feature - Offer Intro

Offer intro is a feature for agents to create, edit and manage their offer introduction. It provides a simple text editor and management functions.

Feature - Property Analytics Dashboard

Property analytics dashboard is designed to help listing agents access property related data in a systematic and automatic way including marketing feedback, days on the market, number of offers and so on. In doing so, listing agents could adjust marketing strategy and pricing strategy accordingly as well as report the data to their clients as a showcase of their work.

Design System - Banners

A banner displays an important, succinct message, and provides actions for users to address (or dismiss the banner)in the app. In the reside platform, banners are widely used to communicates a change or error within the app. I took the responsibility to create the banner design system for the reside app based on Material UI.
